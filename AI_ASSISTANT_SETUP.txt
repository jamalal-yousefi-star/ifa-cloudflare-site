IFA Smart Assistant v30

This version works in two modes:

1) Local smart mode:
- Works immediately without any API key.
- Uses built-in IFA service knowledge and human-like fallback responses.

2) Real AI mode:
- Add OPENAI_API_KEY in Netlify:
  Site settings > Environment variables > Add variable
  Key: OPENAI_API_KEY
  Value: your OpenAI API key
- Optional:
  OPENAI_MODEL = gpt-4o-mini or another compatible model.
- Deploy again after adding the key.

Notes:
- Do not put the API key inside index.html.
- The key is used only inside netlify/functions/ifa-assistant.js.
- If the key is missing or fails, the assistant automatically falls back to local smart mode.
